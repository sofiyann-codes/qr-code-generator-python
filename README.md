# QR Code Generator using Python

A simple and customizable QR Code Generator built using Python.  
This project allows users to generate QR codes for URLs, text, Wi-Fi credentials, and contact cards (vCard).

---

## 🚀 Features
- Generate QR codes for text and URLs
- Create Wi-Fi QR codes (auto-connect on scan)
- Generate contact QR codes (vCard)
- Customize QR size and colors
- Save QR codes as PNG images

---

## 🛠️ Technologies Used
- Python 3
- qrcode library
- Pillow (PIL)

---

## How to Run
1. Install Python
2. Install requirements:
   python -m pip install -r requirements.txt
3. Run:
   python qr_generator.py
